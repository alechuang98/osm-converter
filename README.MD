# OSM CONVERTER

### Dependencies
python 3.6.8
numpy 1.17.5
matplotlib 3.1.3

### Usage
```
python src/Convert.py --input <input json file> --log <log png file> --output <output json file>
```

##### input file format

json format. `test/input.json` for reference.
